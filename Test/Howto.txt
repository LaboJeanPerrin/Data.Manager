=== PNGCRUCH ===============================================================

Note: work only on PNG images. I use the output of Matlab's imwrite.

--- Install

sudo apt-get install pngcrush

--- Usage

 Let pngcruch choose the best filter (long !):

pngcrush -brute -e ".compressed.png" image.png
